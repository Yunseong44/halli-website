## 산출물 목록

### 1. 정적 사이트 전체 소스 코드

**파일**: `halli-website-source.tar.gz` (96MB)

**포함 내용**:
- 완전한 React 프로젝트 구조
- 모든 컴포넌트 및 페이지
- 스타일시트 및 설정 파일
- 의존성 잠금 파일 (`pnpm-lock.yaml`)
- 빌드 설정 (`vite.config.ts`, `tailwind.config.ts`)

**빌드 방법**:

```bash
# 1. 압축 해제
tar -xzf halli-website-source.tar.gz
cd halli-website

# 2. 의존성 설치
pnpm install

# 3. 빌드
pnpm build

# 4. 결과물은 dist/public/ 디렉토리에 생성됨
```

### 2. 빌드된 정적 사이트 (즉시 호스팅 가능)

**파일**: `halli-website-build.tar.gz` (287KB)

- `assets/index-*.css` (116KB) - 스타일시트
- `assets/index-*.js` (558KB) - JavaScript 번들

**사용 방법**:

| 요소 | 폰트 | 크기 | 가중치 |

- **입력**: 반투명 배경, 포커스 시 강조색 테두리
- **토스트**: Sonner로 자동 관리
